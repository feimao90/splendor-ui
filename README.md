# splendor-ui ![Status](https://img.shields.io/badge/status-WIP-blue)

光辉号骁之屋桌面版前端组件库，使用 Vue3 构建。

基于 MIT 协议，可自由修改和使用。

## 在线示例

[https://splendor-ui.ybusad.com/](https://splendor-ui.ybusad.com/)

## 安装

使用 NPM：

```bash
npm i splendor-ui
```

使用 Yarn:

```bash
yarn add splendor-ui
```

## 参与开发

```bash
make install
```

开始调试：

```bash
make dev
```

打包：

```bash
make build
```
